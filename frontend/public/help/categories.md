### Category & Subcategory Master

The Category Master registry dictates the taxonomy buckets used during file ingestion. Setting up structured classifications guarantees search index accuracy.

#### Actions:
- **Create Category:** Use the top input box to initialize top-level folders (e.g., Finance, Legal).
- **Edit Category:** Click the pen icon next to a category name to rename it across all indexed document mappings.
- **Delete Category:** Click the trash icon. Note that deleting a category also cascades and deletes all associated child subcategories.
- **Add Subcategory:** Enter a name in the text box inside the category card and click "Add" to set up nested classifications.
