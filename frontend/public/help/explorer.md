### Workstation Directory Explorer

The Explorer provides a directory tree browser to view how documents are organized physically on your storage disk.

#### Disk Organization Strategy:
- Ingested files are moved into folder nodes structured by extension and ingestion date.
- Path layout: `organized/[fileType]/[year]/[month]/[filename]`

#### Available Actions:
- **Expand Nodes:** Click folders to browse subdirectories.
- **Open Location:** Clicking "Open Location" opens Windows File Explorer with the physical document pre-selected on your system.
- **Run File:** Clicking "Run Local File" starts the document in its native OS application handler (e.g. Acrobat Reader, Microsoft Word, Photos).
